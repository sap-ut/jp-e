# SP-7 Glass ERP

Complete ERP Solution for Glass Processing Industry

## Features
- Proforma Invoice with Excel-like Grid
- Work Order & Production Tracking
- Cutting Optimizer (Nesting Algorithm)
- Fast Process Master (Hole, Cutout, Shape)
- 40+ Additional Charges with GST
- Customer Management (Bill To / Ship To)
- Price Master (Size-wise, Thickness-wise)
- Glass Inventory & Hardware Stock
- 20+ Management Reports
- Tally Export
